“Touch”(Ch-3)

发表于 2021-04-15 更新于 2023-02-24 本文字数： 1.2k 阅读时长 ≈ 4 分钟

今天，我的室友蔡因病被送去医院。我和室友高陪同了一晚上。

七点钟了，蔡的爸爸仍然没来。高和我说：“咱们怕是要翘课了。”

我说：“老师不知道，就是去了。我一会问问点名情况。”

制造交流的一种好办法是：麻烦别人。我能想到的，就是小白了。

九点多的时候，我给她发信息：“小白小白，今天仿生学课点名了吗？”

九点四十五的时候，她才回复：“我一觉睡到现在，听朋友说，有课堂测试耶……糟糕了。”

我就把室友进医院以及我们的状态和她讲了一遍。

“加油！和老师说一声，应该就没事了。”

*“你怎么问她？”*室友高的靠近，让我吃了一惊。

“我今天告诉你一个秘密。”

“什么秘密？”

“因为我喜欢白若静。”

“……”

我们等到他的家长到来，十二点多才回到学校。我们吃了点面包，就去做物理化学实验了。今天的实验非常无聊，只需启动仪器，然后等待，然后从电脑中复制数据。在等待的时候，小白的同桌诸一如既往在闲逛，我借此机会坐到小白旁边，她已经结束了实验，关闭了软件，正准备把电脑关机。

我凑上前去说：“为什么不玩玩扫雷呢？”

接下来的事情，让我有点惊讶。“对，对……” 她取消了关机的动作，熟练地启动了开始菜单中的扫雷游戏。

她灵巧地点击着游戏界面，眼珠随着光标而移动。看到容易的定式，她就带着微笑作上一个个标记；而出现了复杂的情况，她一定会先皱皱眉，随后眉毛又随着问题的解决而舒展开，自然也少不了淡淡上扬的嘴角。

我没有见过多少玩扫雷的人，更别提这么熟练的人，更更别提是这样的一个女孩子。我就在一旁默不作声地偷偷看着她，就像那天在实验室一样。

这时突然出现了一处需要猜的地方。“哎呀，这可怎么办呢？” 她蹙着眉抱怨道。

我在一边说：“猜一下吧！”[1]

*“哈！”*她的运气很好，这样几下，完成了许多处疑惑的布局。

她就这样完成了一局游戏，这台电脑还没有人解开“高级”难度。

『请输入您的尊姓大名』

我又在旁边附和着，“19000481！”[2]

“哈哈，果然永远都是19000481。”

她对我说：“你要玩吗？”

我回答：“不，我看你玩儿就行啦！”

事实上这是因为，如果我开始游戏，她一定会去和那个人说话，而我只能趁着这种难得的机会，和她相距这么近。

我又说：“你玩儿扫雷太强了，我都没打通过高级。”

“我好久没有在电脑上玩扫雷了。我觉得还是在手机上好操作一些。”

我觉得这是插入话题的绝佳机会，“你难道天天熬夜，就是在玩儿扫雷吗？”

“不是不是，我会看看动漫。我昨天好像两点才睡……睡到了九点多钟，所以仿生学都错过了。”

“我现在都熬不动了……身体很差劲。你们宿舍的人都不互相叫一下吗？”

“她们都没有课。”

“哦，只有你是生科的，她们都是生工的。我们宿舍还好，我是起的最晚的。高六点四十就起来，蔡这学期早起跑步，设六点的闹钟，还是搞笑铃声。杨走了，又来了一个当兵回来的室友丁，也是早睡早起。”

“哈哈哈！反正我喜欢熬夜……根本就停不下来。”

“我很羡慕你们宿舍，经常结伴一起出去玩，我们宿舍从来都是在宿舍一呆呆一天。也都不玩游戏，都是玩一些小游戏。”

“女生嘛，就喜欢一起出去逛……”

……

我们聊了很久，我也记不清后来说了些什么。只知道这一天下来，虽然半夜室友出了问题，陪着熬了一夜，但至少回来，发生了这么开心的事情，我的虚荣心得到了极大的满足，尽管最后还是那个人和她走在一起。

今天和她聊天，比不上那个人的一百分之一吧……

老师说，可以走了。大家都收拾着东西准备离开，他们也不例外。

我强忍着不舍的心情和她说：“以后一定要早睡早起，我也不好天天提醒你。”

  2. 这是小白的学号。 ↩︎

“Reconnection”

发表于 2021-03-01 更新于 2023-02-24 本文字数： 851 阅读时长 ≈ 3 分钟

假期没有发生什么。我唯一记得的一件事，就是她在公历2月14日，这个奇怪的日子，发了这样一条动态：

“初三快乐。”[^1]

这让我觉得，之前所看到的一切，或许是假象。于是下一学期的打算，又成了“变成那样的人”。

到了新学期，一切都没什么变化。但我在周一的第一次课看到小白和那个人拉着手时，我又开始难过起来了。我想这学期努力，或许会成功。

这学期的第一个月，我所做的，就是能和她打招呼时就制造一些交流。到了第二个月，是一场突如其来的变故，或者说是一直存在的变故，让这发生了将近一年的短短记忆结束了。

这是这学期的第一节课，生物化学。如同上学期的生理学一样，生科、生技两个专业一起上课。不过由于同时开设全英文课程的缘故，自然少了许多或是熟悉或是陌生的面孔。

我难以评价去上全英文课程的同学。或许他们是真的想开阔视野，提高自己的水平；或许他们是为了避免早起。我只能说我是一个认得清情况的中庸之辈。

看看某些熟悉的面孔。熟悉的团支书于同学打扮得似乎更加成熟感性、头发的波浪更大了；熟悉的刘同学刘海还在，不过长长的头发也多出了波浪。我其实没有注意到她，是室友指给我的。当时我们感叹：

“你看看你的老朋友。”

我看了一眼，有些吃惊。接着说：“我的青春结束啦！”

“你再看看她们。”

我向着他手指的方向看了一眼，有两个叫不上名字的女同学，她们一个把头发染成了纯蓝色[^1]，另一个则是鲜艳的紫色[^2]。我不由得吃惊了。

“那看来我的‘老朋友’还是一如既往，至少看起来绝对是个正常人。”

……

而小白在哪呢？我没有看到她，或许在我们后面的位置和男朋友在一起度过着愉快的时光吧？如果上学期我和她的接触算是浅尝辄止，那么我也就只能做到浅尝辄止，未来的路，若无必要，不再相拥，也难以再相拥。

这之后我果然看见他们走在一起。上学期还只是一起说说话一起走走路，这学期已经是大手拉小手。

我可能过于平凡地看待了大学的生活，恋爱已然不是一件讳莫如深的事情，但我的思想仍然迂腐。我还是在美好事物彻底离开后才潸然落泪。

下午的免疫学课如同上学期的生态学，看着熟悉的教室结构，我越发想起疫情后第一次见到小白的事情。不过人明显少了很多，但我这次坐在最后一排却仍然不见小白。他们八成是去了英文班。我如何思索？可能那个人会带着她一起进步，一起攀向高处，而我却自顾自地望着黑暗。

“Touch”(Ch4)

发表于 2020-12-25 更新于 2023-02-24 本文字数： 460 阅读时长 ≈ 2 分钟

这一年，与她相遇的最后一天，是那次生态学考试，也是那次圣诞节。那一天的阳光温暖而不刺眼，是我度过的最快乐的圣诞节。为何我会如此快乐？忘了告诉你，小白是一个在圣诞节出生的孩子。她不像传说中这一日诞生的神明一样高不可攀，而是与圣诞老人有着同样的亲和力。

我的心情又开始躁动了起来。生态学考试前，我把柜子里仅存的一块草莓味巧克力塞进了口袋里。

考试前，我在考场张望。她不久后从后门走了出来，我也跟着出去了。我在二楼的大平台一如既往和人们谈笑风生。这让我想到了高三的一天一天[^1]，目的尚且一致。很快她便从另一头走了回来，我把她拦住。

她见了我就说，“对不起，我没有带钥匙过来……”

我打断了她，“先别提钥匙的事了。我祝你生日快乐……”一边拿出身上的巧克力。

“谢、谢、你～” 她的笑容与她手中的草莓味巧克力散发着同样的气息。

这之后，我便如常回到教室。尽管我并没有掌握许多知识，但想着刚才的美好，也带着自信完成了考试。

考试结束后，就放寒假了。回想着这学期的一点一滴，我觉得再这样下去，可能会出现不必要的麻烦。我在假期，克制着自己的心情，没有打扰她。

但这样的心情，并不是短短的两个月就能消除的。所以我下学期，所面对的只能是冷冰冰的事实。就让冰冷的事实，让我清醒起来吧。

“Touch”(Ch2)

发表于 2020-12-02 更新于 2023-02-24 本文字数： 337 阅读时长 ≈ 1 分钟

有一天的遗传学实验是“群体遗传分析”，要观察自己的指纹和掌纹。老师发下白纸和印泥，要我们把手印在纸上。等候同桌的一段时间里，我随意地走动，看见她涂了很多印泥在手上，变得红红的。我想要开玩笑，说：

“杀人犯！”

她咯咯地笑。“明明没有啦……”

她有着奇妙而悠扬的嗓音，那声音似乎伴随着欣喜。隔着口罩我也能看到她的笑容，笑得傻而天真。

之后又继续果蝇的实验。她走到边上去准备乙醚。我也凑了上去，

“杀人犯把我迷倒啦！”

她这次什么也没讲，只是还在笑着，一双眼睛眯成了两条线。

下课后我就站在门外。她拉着我的手摇摇晃晃地和我一起走进了电梯，一步一步穿过幽静的树林，跨过弯弯的小桥……

可这么做的实际上是另外一个幸运的男同学。你没有看错，我虽然开始试着伸出手，但之前却浑然不知名花有主的事实。我所能预知的结局就是，她真正地成为了一个杀人犯，把我慢慢地抹除。

每周的实验课，就这么不温不火地过去。

“Touch”(Ch1)

我给她介绍了我的小车。“这家伙虽然刹不住车，残破不堪，但你用还是能够满足的。我和你说，车子之后都无所谓，你这个人一定不要因为它出问题了。”

在她思考之际，我突然把看着自行车的头转过来，注视起了她。“我不心疼车子，但我心疼你。”至今我仍不明白，我为什么会这么说，或许是我的一种本能。

“没关系的，只要好用就是好车呀！谢谢你！”

“拜拜，骑着玩去吧！”

我和小白的第一次接触就这么发生了。

那时，和她的交往，还是在实验课堂上比较多。随着时间的流逝，我开始敢于在实验课上的闲暇时间和她开一些玩笑。我不知什么时候已经开始对每周的实验课抱有期待。这个神奇的女孩子闯入了我的生活，或者说，我让自己的世界开始容纳了这个神奇的女孩子。

“Reflection”

发表于 2020-11-22 更新于 2023-02-24 本文字数： 662 阅读时长 ≈ 2 分钟

2019年的9月，是我刚刚上大学的日子。在这一月，我走进了新的环境，认识了新的同学，其中就包括这个姑娘。在第二次生物学实验下课后，她主动地添加我为好友，而我在短暂的寒暄过后，竟然稀里糊涂地没有和她保持联系，于是就有了一年后各种事情的发生与终结。

重新和她认识是在一年多后的遗传学实验。或许是运气好，或许也是运气差，我和她都选择了分流到生物科学专业，仍然是同班同学。实验课时我和她的座位也相距很近，常常聊天。

这位姑娘，来自南方。或许是因为她姓白，她就被大家叫做小白，并且由于她以这种称呼自称过自己，所以我也习惯了这样的称谓。她的家乡在遥远的云南，是一位少见的彝族姑娘。来到这里求学的她，也成了我能够观赏的彩云之南的风景。

我刚认识小白这个人的时候，她还梳着不到肩膀的短发。待疫情回来之后，我发现她的头发被留长了。有时她会简单地把它们扎起来，有时会梳着一个短短的麻花辫。如果是随意的时候，就会把它们披下来。她的五官起伏比较大，所以那双带些褐色的眼睛看起来大而有神，但她总会戴一副圆眼镜，这就显得眼睛小些。她的眉毛也很明显，而化妆时也不喜欢浓抹，这一张脸的亮点就完全突出在了她的眼睛上。正如“皎洁圆明内外通，清光似照水晶宫。”

我唯一记得的实验课前见到她的两个时间点，第一个是，我冒着雨赶向开班会的教室，很偶然地坐到了她的旁边。我忘记了她的打扮，只记得她好像梳着那个麻花辫，十分可爱；第二次，则是某次毛概课前，我在E教学楼的走廊看到她和新班级的一个男同学走在一起。

而课下的重新接触，是有一次，她在动态里抱怨，天气太冷，还要进行晨跑而吃不消。我在她的动态里评论，“小白加油，我可以借你自行车。”过了几天后，她在深夜真的来找我，提起了这个请求。第二天线性代数下课，我与她相约在贩卖机旁见面。

“Again, Your Birthday”

发表于 2020-11-05 更新于 2023-02-24 本文字数： 427 阅读时长 ≈ 2 分钟

明天又是陈小姐的生日，我还决定，用简简单单的一段话来祝福她，进行一次诀别。

无物感恩，唯借只纸片言。我没有什么可送你的，只能简简单单和你说声生日快乐，同时也感谢在那一年你在我生命中的存在，也许日后我们还是像现在一样在相隔很远很远的地方各自努力，尽管时光不复返，但我会努力守护那段在一起的记忆；也可能会是在某个更自由的地方再续缘分，那样便是来日方长，后会有期。

“好，谢谢你啦，也祝你快乐。”

我一直喜欢的，似乎是“陈同学”，而非现在的“陈小姐”。自己想想，从小到大，我似乎总是对那些“不漂亮的女孩子”感兴趣。然而陈小姐现在与我的理想在渐行渐远吧！她在慢慢地变。她在从青涩可爱慢慢地变得成熟。而我还是对那个穿着红色衬衣和校服向我微笑问好的陈同学恋恋不舍。如果我早一点加上她的微信，关注她，可能就会早早意识到这个事实。然而人总是会不由自主地对过去的事情进行假设。我很感谢陈同学为我做出的一切，哪怕只是在那平庸的一年出现过。日后只要坚信，陈同学为我带来了美好的一年，之后她便选择去四处闯荡，变成了离我好远的陈小姐。

两年时光，就此结束。

谢谢你，我要去寻找自己的未来啦。

“L”

发表于 2020-10-30 更新于 2023-02-24 本文字数： 702 阅读时长 ≈ 3 分钟
  • 本文完成时想以故事结束时间为文章时间，但时间久远，已经忘记，暂时设置为一个不影响其他文章逻辑的时间。

L同学是我大一时的同学，但当时我没有注意到她，因为当时她看起来很普通甚至可以说比较丑陋。而大二时我才注意到她，那时是在补做分析化学实验，她正对着我，她虽然仍然是普普通通，但多了梳着的刘海，看起来就是十分让人着迷。当时日记是这样写的：“周五下午补做分析化学实验，看到了许多大一同学，其中有一个女孩子，发际线很高，之前一直简单地把头发束着，今天见到时却梳了刘海，她不是很美丽，但悄悄改变发型后，却更加可爱了起来。虽然我和她并不熟悉，但我却有些心动。我一直觉得，虽不同于一般的涂脂抹粉，但在我眼中这样简简单单的打扮就是最动人的。”

我对她很是心动，但我们已经不是一个班了。我的想法是找某个朋友打探一下，得到的答复是，她一直单身，我的朋友甚至可以帮我。我大受鼓舞，鼓起勇气加上了她的联系方式。开始觉得，她还算是一个有趣的人，并不像常见的女生那样只会嘻嘻哈哈。她给我的感觉是成熟稳重，像个大姐姐一样；她的声音带着沉稳和吸引力。

我正常地靠近她、了解她，但结果仍然不尽人意。在我开始骚扰她近一周后，她称：“我觉得我应该和你好好聊聊，我对你没有那方面的感觉。”我沉默了，看上去普普通通、平易近人的一个孩子，背后却是一个不知多高多深的灵魂。

她有时穿着带着简单卡通图案的衣服，连带着她的发型，像个天真的小孩子；她还有一套白色里夹杂着彩虹颜色的毛衣、配着白纱的长裙和一双小白鞋，那时她就像是天上的仙子。如同她的装束，她就像是朴素的天边出现的异色的彩虹，真的十分独特，令人神往。

虽然这场简单的情愫持续了至多算是两周，但至少让我把待家半年丢失的状态找了回来。也许在不久的未来，我还能找到这样一个普通的姑娘，她有着普通人的脚踏实地和仙子般的吸引力，让我的道路不再那么磕磕绊绊。
